# Blog site with gatsby

hosted by Netlify

## How to run

```
$ cd your-project-name
$ yarn install
$ gatsby develop
```

Running `gatsby develop` you will see the following URLs:

```
http://localhost:8000
http://localhost:8000/___graphql
# Netlify CMS page
http://localhost:8000/admin
```
